
# 响应式网站
	
	这是用HTML+CSS做的响应式网站，根据不同的显示器大小显示不同的页面，加载不同的图片资源。

## 安装开发环境：
    npm install

## 压缩打包代码：
    gulp

    执行gulp 之后 项目会补打包到 dist 目录，代码要放服务器上运行。
